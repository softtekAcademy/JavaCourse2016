What is 2-way binding?


The automated synchronization of data between the MODEL and the VIEW. If MODEL changes, VIEW changes. If VIEW changes, MODEL changes.
The automated synchronization of data between the MODEL and the VIEW. If MODEL changes, VIEW does not change. If VIEW changes, MODEL changes.

What is an expression?


Angular specific constructs that access model variables and functions and that are evaluated within double curly braces {{}}
HTML specific constructs that compute data

What is ng-model?


Angular specific directive that associates an HTML control with a specific property in the model to implement 2-way binding between the HTML control and the model property.
HTML attribute for declaring an HTML element as a function in the controller.

What is ng-bind?


An angular specific directive that is equivalent of an expression in curly braces. It will read a property from the model and will place it in the view.
Angular specific directive that associates an HTML control with a specific property in the model to implement 2-way binding between the HTML control and the model property.
