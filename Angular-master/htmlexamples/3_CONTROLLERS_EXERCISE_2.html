DESCRIPTION:

This example will print a note after clicking either "Chili" or "Custom spice" buttons. The "Chili" button will always print the same note.
The "Custom spice" button will take whatever you inputted in the input text box, will update the model property in the scope with this value, and will print the note using this value.

Important thing to notice:
1. The controller provides the spicy(spice) function that receives a paramter that is used to print the note;

The food is {{spice}} spicy!
