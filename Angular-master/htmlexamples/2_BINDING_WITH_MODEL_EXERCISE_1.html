TEST DESCRIPTION:

This tests reflect 2 way binding between an expression and a variable in the model within the scope. The variable "spice" in the model is updated by the functions defined in the controller and that are invoked when clicking on the buttons. Notice that both buttons invoke the controller functions by utilizing the "ng-click" angular directive that receives as argument the name of the function to be executed upon clicking on them. Every time the "spice" variable changes value Angular automatically updates the view (HTML page) with the new value. This feature of automatically updating the view (HTML page) when the counterpart model changes is called 2-way binding.

The food is {{spice}} spicy!
