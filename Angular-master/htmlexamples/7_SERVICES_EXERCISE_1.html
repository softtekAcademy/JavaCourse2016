DESCRIPTION:

This example registers a service called "notify" using the factory API at the module level. It receives as a dependency the $window angular-specific service. The service will launch an alert window to the user if the service is invoked 3 times. The alert will be thrown using the $window angular-specific service.

Note: Angular-specific $window service documentation can be found at https://docs.angularjs.org/api/ng/service/$window.
