DESCRIPTION:

This exercise shows how to use 5 directives: ng-app, ng-init, ng-repease, ng-model, & ng-if.

  ng-app: Associates an HTML element with an angular application module.
  ng-init: Initializes model properties in the scope.
  ng-repeat: Use to loop to a collection of objects.
  ng-model: Associates an HTML element with a model property.
  ng-if: Do something if expression associated with ng-if directive is true.

NOTE: Notice that this exercise uses ng-app without any reference to any angular module. This is correct. ng-app takes an optional argument which is the module name. If none is provided, the ng-app by itself lets Angular knows that every HTML under the ng-app directive needs to be treated as an angular application.
