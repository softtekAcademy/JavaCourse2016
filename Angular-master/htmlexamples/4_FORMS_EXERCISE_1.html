DESCRIPTION:

This example show a group of controls that compose a Form. Notice that the form's controls (input text boxes and radio buttons) make use of the ng-model directive that tells angular that these controls are tied to the model using an object called "user".

Important thing to notice:
1. The "novalidate" word a the form level disables native browser's form validation. That is not necessary when working with angular provided angular controls provide validation.
2. The value of ngModel won't be set unless it passes validation for the input field. For example: inputs of type email must have a value in the form of user@domain.
