DESCRIPTION:

This example show a group of controls that compose a Form. Notice that the form's controls (input text boxes and radio buttons) make use of the ng-model directive that tells angular that these controls are tied to the model using an object called "user".


Important thing to notice:
1. Check that we have customized 2 of the CSS classes used by angular automatically. When the control in the form has been touched and it has failed validation (i.e. in case of input control, invalid will be an invalid email) then the style we defined will paint the control in red color. This is the style that was implemente: .css-form input.ng-invalid.ng-touched . Similarly, if the control has been touched and it is valid, this .css-form input.ng-valid.ng-touched will paint the control in green.
2. In angular, these classes are set automatically to our controls. We can modify the look and feel by just ensuring that our pages define the stlyes.
