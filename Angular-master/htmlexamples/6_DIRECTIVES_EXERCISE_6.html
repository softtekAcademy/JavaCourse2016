DESCRIPTION:

This exercise defines 2 directives that redirects to the same template URL file. One of the directives was created as an Element only directive using restrict: 'E' option, and the other one was created as an attribute directive using the restrict:'A' option.

NOTE: Notice that the directives are being used 2 times each. One of the usages is correct, the other one is not correct. When the directive is incorrectly used it will not print. We need to use them correctly pero their definition (restrict:'E' directives will only work if used as an HTML element, restrict:'A' directives will only work if used as an attribute of an already defined HTML element.})
NOTE: Notice that the directives are using the scope of the controller. There is an option to define a directive to have its own scope created. We will review this in the next exercise.
