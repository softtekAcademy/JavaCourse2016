DESCRIPTION:

This exercise shows how to use 3 directives: ng-app, ng-show, & ng-hide. It is a must. It tells angular that anything under this directive is an angular application.

  ng-app: Associates an HTML element with an angular application module.
  ng-show: Shows an html portion when the expression associated with the ng-show directive is true.
  ng-show: Hides an html portion when the expression associated with the ng-show directive is true.

NOTE: This exercise makes use of the optional angular library called angular-animate.js. All angular optional libraries can be found under https://code.angularjs.org/1.4.3/. Remember that the core library used to run angular is angular.min.js.
NOTE: This exercise is loading the ngAnimate optional module using ng-app.
